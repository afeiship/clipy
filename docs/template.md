# clipy
> Pbcopy for nodejs.

[![version][version-image]][version-url]
[![license][license-image]][license-url]
[![size][size-image]][size-url]
[![download][download-image]][download-url]

## installation
```shell
git clone https://github.com/afeiship/clipy.git
cd clipy
npm i && npm link
```

## usage
~~~
clipy hello
clipy `pwd`
~~~
